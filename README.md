# go-shop
## Run with go
1. Install [Go](https://golang.org/doc/install).
2. Install [MongoDB](https://docs.mongodb.com/manual/installation/).
3. Run MongoDB:
3. Run app:
```
  go run main.go
```
## Run with docker compose
1. Install [Docker](https://docs.docker.com/get-docker/).
2. Run app:
```
  docker-compose up -d --build
```
